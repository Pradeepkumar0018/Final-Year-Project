# Emotion-Based Music Recommendation System

## Overview
This project detects a user's emotion from voice input and recommends songs based on the detected emotion.

## Features
- Voice Recording
- Audio Feature Extraction
- Emotion Detection
- Music Recommendation

## Technologies Used
- Python
- Machine Learning
- Librosa
- Scikit-learn
- Flask

## How to Run
1. Install requirements
2. Run app.py
3. Record voice
4. Get recommendations
